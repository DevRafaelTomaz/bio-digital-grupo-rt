# 🌐 Link na Bio – Grupo RT

Este projeto é um site pessoal estilo “cartão de bio” para Instagram, criado com foco em identidade profissional, criatividade e integração com redes sociais.

---

## 🤖 Desenvolvido 100% com Inteligência Artificial

Nenhuma linha de código foi escrita manualmente.  
Todo o conteúdo, layout, estilos e estrutura foram gerados com **Inteligência Artificial**, utilizando duas ferramentas principais:

- **ChatGPT (OpenAI)** → Para criação dos prompts, organização técnica do projeto, README e estrutura lógica
- **Claude (Anthropic)** → Para desenvolvimento do layout visual, design responsivo e sugestões estéticas

> “Eu não programei. Apenas escrevi o que queria — a IA fez o resto.” — *Rafael Tomaz*

## 🚀 Funcionalidades

- Layout responsivo (mobile-first)
- Visual moderno, inspirado em cartões digitais criativos
- Avatar circular com nome e título profissional
- Botões personalizados com ícones de redes sociais
- Rodapé com a identidade visual do grupo

---

## 🛠️ Tecnologias Utilizadas (Geradas por IA)

- **HTML5** – Estrutura da página
- **Tailwind CSS** – Estilização moderna, leve e flexível
- **Font Awesome** – Ícones para botões de redes sociais
- **Design IA** – Layout, espaçamento, responsividade e animações criados via IA

---

## 💡 Inspiração Visual

O layout foi inspirado em páginas como a de **Yasin Ahmed** e outros criadores que utilizam um estilo minimalista e funcional para sites de bio digital. O objetivo é entregar uma experiência leve, atrativa e profissional para visitantes que chegam pelo link do Instagram.
